== Description ==

Blogs Perú Ping, es un plugin que funciona bajo WordPress 2.0.x o superior, el cual sirve para enviar pings de manera automática a blogsperú cada vez que publiquemos una nueva entrada o editemos alguna entrada.

* PHP 4.1 (o mayor) 
* Soporte para sockets 
* WordPress 2.0.x o superior

== Installation ==

- Desempacar el Archivo en un directorio vacio.
- Abre con cualquier editor el archivo "blogs_peru_ping.php".
- En la línea 19 se encuentra la variable a cambiar, colocas el ID que le corresponde a tu blog y listo eso es todo.
- Cómo averiguar tu ID en blogsperu.com:
  Ingresa a esta dirección: http://www.blogsperu.com/ping/default.asp, ingresas tu nombre y direccion URL de tu blog, en la siguiente
  página vas a visualizar un link con el texto "HACER PING AUTOMATICO" le das click y en la direccion de tu browser vas a visualizar algo parecido a esto:
  http://www.blogsperu.com/ping/auto.asp?id=3063 como veras el ID en este ejemplo es: "3063" .

- Subir el archivo guardado al directorio "/wp-content/plugins/". 
- Entra a tu panel de administracion o dashboard y activa el plugin.
- Listo ahora cada vez que publiques o edites una entrada, sera notificado de manera automática blogsperu.com .

== Frequently Asked Questions ==

= Cómo averiguar tu ID en blogsperu.com =
1. Ingresa a esta dirección: http://www.blogsperu.com/ping/default.asp
2. Ingresas tu nombre y direccion URL de tu blog
3. En la siguiente página vas a visualizar un link con el texto "HACER PING AUTOMATICO" le das click.
4. En la direccion de tu browser vas a visualizar algo parecido a esto: http://www.blogsperu.com/ping/auto.asp?id=3063 como veras el ID en este ejemplo es: "3063".

= Como edito el archivo blogs_peru_ping.php =

1. Abrir el archivo en un editor de texto.
2. Ubicarse en la línea 19.
3. Cambiar la variable que aparece ahi con el ID que te proporciona blogsperu.com.
